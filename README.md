# ai-mentor-chatbot
AI Career Mentor Chatbot

An intelligent chatbot that helps students explore careers in Artificial Intelligence, Machine Learning, and Data Science.
It provides learning roadmaps, interview preparation tips, and AI project suggestions using Large Language Models and Retrieval-Augmented Generation (RAG).

Live App

https://ai-mentor-chatbot-nnaq4vphv2ybbiksvnkshy.streamlit.app/

Features

- AI career guidance
- Retrieval-Augmented Generation (RAG) using local documents
- AI learning roadmap suggestions
- Machine learning interview preparation
- Concise and detailed response modes
- Optional web search integration

Technologies

- Python
- Streamlit
- Large Language Models (LLMs)
- RAG (vector embeddings)

Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py

Example Questions

- How do I become an AI engineer?
- What AI projects should I build?
- Explain bias vs variance.

This project was developed for the NeoStats AI Engineer Challenge.
